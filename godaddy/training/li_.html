  • Publications

  • - Coding Analog sensors on the Raspberry Pi3

    CodeGuru, Sep 2017.

    This is about to make raspberry pi 3 interfacing with Analog sensors through ADC
    mcp3008 chip using SPI protocol way of communication. Sensors covered are
    a)JoyStick
    b)Potentiometer(10k)

  • - Coding sensors on the Raspberry Pi3

    CodeGuru, Sep 2017.

    Discuss primarily digital sensor communicating with rPI3. Light and Moisture sensor
    analog signal is captured through Analog to Digital Converter (ADC) pcf8591,
    communicates using I2C. Sensors covered are
    a) LED (digital)
    b) DHT11, humidity and temperature sensor (digital)
    c) Motion PIR sensor (digital)
    d) Light sensor, digital and analog
    e)Moisture sensor, digital and analog

  • -Raspberry Pi3 hardware and system reference

    CodeGuru, Sep 2017.

    Discuss about rPI3 interfacing protocol with external hardware. Protocols include
    I2C and SPI. Protocols have been implemented in user space with mapping pysical
    memory /dev/mem to virtual address space and real time register bit tweaking.
